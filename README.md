# ng-tasks
Front-end and back-end for scrum tasks


Run Front-end

```sh
$ cd ng-tasks
$ npm install
$ npm start
```

Open http://localhost:4200

Back-end
```sh
$ cd node-tasks
$ npm install
$ nodejs bin/www
```

Back-end is on the port 3000




You will see 4 tasks for each column, you can add and remove tasks.
